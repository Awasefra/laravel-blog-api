# Installations

## Clone repositories

```
git clone https://github.com/Awasefra/laravel-blog-api-with-redis.git
```

## Create & update env File

```
cp .env.example .env
```

###

## Generate key

```
php artisan key:generate
```

## Install resourse laravel

```
composer install
```

## Launch Apps

```
php artisan serve
```
